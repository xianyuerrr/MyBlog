# 异或

异或运算的性质：

**1.** 交换律：A ^ B = B ^ A;

**2.** 结合律：A ^ (B ^ C) = (A ^ B) ^ C;

**3.** 恒等律：X ^ 0 = X;

**4.** 归零律：X ^ X = 0;

**5.** 自反：A ^ B ^ B = A ^ 0 = A;

**6.** 对于任意的 X： X ^ (-1) = ~X；

**7.** 如果 A ^ B = C 成立，那么 A ^ B = C，B ^ C = A；

**8.**$\forall i \in Z$，有 $4i \oplus (4i+1) \oplus (4i+2) \oplus (4i+3) = 0$。
